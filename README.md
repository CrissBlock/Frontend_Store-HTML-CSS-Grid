# 🛍️ Frontend Store

Tienda online de productos para desarrolladores, desarrollada con HTML5 y CSS3. Incluye grilla de productos, página de detalle y sección sobre nosotros. Diseño responsive con CSS Grid y Flexbox.

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3** — Grid, Flexbox, Variables CSS
- **Google Fonts** — fuente Staatliches

---

## 📄 Páginas

- **Tienda** — grilla de productos con imágenes y precios
- **Producto** — página de detalle con formulario de talla, cantidad y botón de compra
- **Nosotros** — descripción del negocio y sección de beneficios

---

## ✅ Funcionalidades

- Grilla de productos responsive con CSS Grid
- Imágenes decorativas posicionadas con `grid-row` y `grid-column`
- Página de detalle de producto con selector de talla y cantidad
- Sección "¿Por qué comprarnos?" con íconos y descripción
- Enlace activo resaltado en la navegación según la página actual
- Diseño responsive adaptado a móvil y desktop
- Variables CSS para colores y tipografía
- Nomenclatura BEM en todas las clases
